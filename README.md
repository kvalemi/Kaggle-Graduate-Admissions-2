## Kaggle Graduate Admissions 2

This project focused on the Kaggle dataset called "Kaggle Graduate Admissions 2" posted in the following link:

https://www.kaggle.com/mohansacharya/graduate-admissions

I performed exploratory data analysis on the dataset and built a simple regression tree for prediction of admission probability. The full analysis can be found either in the RMD file or the compiled HTML report.
